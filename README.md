### features 
* QT(pySide)로 작성되어 MS-windows 에서 작동 
* AV2의 "xls파일로 저장" 기능을 통해 기간별로 추출
* 추출 한 내용을 CVS로 변환 -> 공격유형별로 구분 
* 간단한 리포트를 txt 파일의 형태로 저장 기능
